# pyType

## Tentang Proyek

**pyType** adalah proyek latihan mengetik kode untuk membiasakan penulisan syntax pemrograman.

## Menjalankan Proyek

```sh
git clone https://github.com/idrislaode/pytype
cd pyType
npm install
npm run dev
```

Buka alamat yang muncul di terminal melalui browser.

## Teknologi

* Vite
* React
* TypeScript
* Tailwind CSS

## Tujuan

Melatih kecepatan dan ketepatan mengetik kode.
